# Applied projects

Applied projects combine several course modules before the final capstones.

- [`tasks/`](tasks/README.md) — one Task contract with SQLite and Markdown
  persistence, Axum and Actix Web servers, a Reqwest client, and a shared CLI.

The Task project is intended after
[Module 13](../lessons/13_rest_apis_and_http_clients/README.md) and before
[`../capstones/`](../capstones/README.md). Its starter and solution are separate
workspace packages so learners can compile either boundary explicitly.
